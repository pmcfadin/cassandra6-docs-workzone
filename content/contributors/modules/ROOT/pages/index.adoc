= Contributing to Cassandra

[IMPORTANT]
====
*Preview | Unofficial | For review only*
====

Apache Cassandra is an open-source project that welcomes contributions of all kinds: code, documentation, tests, bug reports, and tooling improvements.
This section provides everything you need to go from first clone to committed patch.

== Where to Start

If you are new to the project, start with xref:build-test/gettingstarted.adoc[Getting Started] to set up your environment, then read xref:patch-review/patches.adoc[Contributing Code Changes] for the contribution workflow.
